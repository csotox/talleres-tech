# Curso MEVN en Faztweb

Tecnologías:

- MongoDB
- Express
- Vuejs
- Nodejs

## Estructura de carpetas

- /src (Directorio principal de nuestra aplicación)
  - /app (Archivos del FrontEnd)
  - /models (Modelos de MongoDB)
  - /public (Carpeta publica en la web)
  - /routes (Rutas del servidor)
